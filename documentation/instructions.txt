To create an interactive directory tree structure in `index.html` that allows users to navigate through the Flask and Node.js server directories, you can utilize HTML, CSS, and JavaScript. Below, I'll provide a basic implementation that meets your requirements.

### Directory Structure Setup

Assuming the following structure, we will create a tree view for the `my_project` folder:

```
/my_project
│
├── /flask_server
│   ├── __init__.py
│   ├── server.py
│   ├── /templates
│   │   └── index.html
│   └── /static
│       ├── /css
│       │   └── styles.css
│       └── /js
│           └── scripts.js
│
├── /node_server
│   ├── server.js
│   └── package.json
│
└── /logs
    ├── flask.log
    └── node.log
```

### Step 1: HTML Structure

Here’s the `index.html` file, which will serve as the interactive directory tree.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Directory Explorer</title>
    <link rel="stylesheet" href="../static/css/styles.css"> <!-- Adjust the path as necessary -->
    <script src="../static/js/scripts.js"></script> <!-- Adjust the path as necessary -->
</head>
<body>
    <h1>Directory Tree Explorer</h1>
    <div id="tree"></div>
    <script>
        // Sample directory structure data
        const directoryStructure = {
            "my_project": {
                "flask_server": {
                    "__init__.py": null,
                    "server.py": null,
                    "templates": {
                        "index.html": null
                    },
                    "static": {
                        "css": {
                            "styles.css": null
                        },
                        "js": {
                            "scripts.js": null
                        }
                    }
                },
                "node_server": {
                    "server.js": null,
                    "package.json": null
                },
                "logs": {
                    "flask.log": null,
                    "node.log": null
                }
            }
        };

        // Function to create the tree structure
        function createTree(container, structure) {
            for (const key in structure) {
                const isDirectory = typeof structure[key] === 'object';
                const item = document.createElement('div');
                item.className = isDirectory ? 'directory' : 'file';
                item.textContent = key;

                if (isDirectory) {
                    const childContainer = document.createElement('div');
                    childContainer.classList.add('child-container');
                    childContainer.style.display = 'none'; // Initially hidden

                    item.addEventListener('click', () => {
                        if (childContainer.style.display === 'none') {
                            childContainer.style.display = 'block';
                            createTree(childContainer, structure[key]);
                        } else {
                            childContainer.style.display = 'none';
                        }
                    });
                    item.appendChild(childContainer);
                } else {
                    item.addEventListener('click', (event) => {
                        event.stopPropagation();
                        alert('File clicked: ' + key);
                        // Fetch the file or handle accordingly
                    });
                }
                
                container.appendChild(item);
            }
        }

        // Generate the directory tree on page load
        const treeContainer = document.getElementById('tree');
        createTree(treeContainer, directoryStructure["my_project"]);
    </script>
</body>
</html>
```


### Explanation of the Code

1. **Directory Structure**: The `directoryStructure` object represents the file and directory structure. You can modify it if the structure changes.

2. **Creating the Tree**: The `createTree` function is responsible for dynamically creating the DOM elements for the directory structure. It recursively generates child directories.

3. **Interactions**:
   - Clicking on a directory toggles the visibility of its children.
   - Clicking on a file alerts the user with the file name (you can replace this with logic to fetch the file or perform other actions).

4. **CSS**: Basic styles to differentiate between directories and files, enhancing usability.
