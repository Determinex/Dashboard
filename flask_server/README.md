# BMS
Bookmark Management System

An offline, no-install, private bookmarks manager that can be ran from a flashdrive for situations that involves the use of multiple devices without the need to rely on installations and an Internet connection. 

User data stays with the user. I have no need for your information. 

It is self contained in its own directory. It can import currently saved bookmarks from browsers already in use, create a master list of bookmarks, and allow exportation of this list to centralize your bookmarks across multiple web browsers.

Program Defitions:
-------------------------------------
Bookmarks allow you to store site URLs for quicker viewing later.
-------------------------------------
Bookmark Managers allow you to organize bookmarks with folders and tags, by allowing you to view, add, and remove bookmarks, and it usually features functions for importing and exporting a list of bookmarks, that you then can use with various web browsers on your mobile device or PC.
