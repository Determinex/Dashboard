### Complete JavaScript Code

Here’s the JavaScript code you'll need, including the `toggleNav` function which handles the opening and closing of the slide-out menu:

```javascript
<script>
  // Default search engine URL
  var searchEngine = 'https://www.google.com/search?q=';

  function setSearchEngine(engine) {
      searchEngine = engine;
  }

  function openSearch() {
      var query = document.getElementById('searchQuery').value;
      var url = searchEngine + encodeURIComponent(query);
      window.open(url, '_blank'); // Open the search in a new tab
      return false; // Prevent form submission
  }

  function toggleNav() {
      var sidenav = document.getElementById("mySidenav");
      // Toggle the side navigation menu by changing the left property
      sidenav.style.left = (sidenav.style.left === "0px") ? "-250px" : "0";
  }

  function updateTime() {
      var now = new Date();
      var days = ['Sunday', 'Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Saturday'];
      var day = days[now.getDay()];
      var date = now.toLocaleDateString();
      var time = now.toLocaleTimeString();
      document.getElementById('time').innerHTML = `${day}, ${date}, ${time}`;
  }

  function initializeTime() {
      updateTime();
      setInterval(updateTime, 1000); // Update the time every second
  }

  $(document).ready(function() {
      // Initialize tooltips
      $('[data-toggle="tooltip"]').tooltip();
      initializeTime(); // Start updating time
  });
</script>
```

### Explanation of the JavaScript Functions

1. **`setSearchEngine(engine)`**: This function changes the current search engine if needed (it's not directly called in the provided HTML, but available if you want to implement switching engines).

2. **`openSearch()`**: This function retrieves the search query entered by the user and builds a URL to open the selected search engine in a new tab.

3. **`toggleNav()`**: This function controls the visibility of the side navigation. If the `left` style property of the `sidenav` is `0px`, it moves it out of view by setting `left` to `-250px` (the same width as the menu). Otherwise, it sets `left` back to `0px`, revealing the menu.

4. **`updateTime()` and `initializeTime()`**: These functions manage the display of the current date and time, updating every second.

5. **`$(document).ready(function() { ... })`**: This section ensures that the tooltip functionality of Bootstrap is initialized when the document is fully loaded, and the time display starts updating.

### Final Integration

Make sure you include the JavaScript code at the bottom of your body section, just before the closing `</body>` tag, as shown in the previous example. This is important to ensure that the DOM is fully loaded before any scripts run.
